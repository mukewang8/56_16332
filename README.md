<h3>课程介绍：</h3>
<p>模版元编程（template metaprogram）是C++中最复杂也是威力最强大的编程范式，它是一种可以创建和操纵程序的程序。模版元编程需要很多技巧，常常需要类型重定义、枚举常量、继承、<a title="查看与 模板 相关的文章" target="_blank">模板</a>偏特化等方法来配合，因此编写模版元编程比较复杂也比较困难。</p>
